# Git & GitHub Learning Roadmap

This roadmap is designed to take you from an absolute beginner to a confident Git user. Follow these steps in order.

## Step 1: The Basics (Week 1)
- [ ] Read through `01-introduction`
  - Understand Version Control, Git, and GitHub.
- [ ] Complete `02-setup`
  - Install Git, configure your profile, and set up SSH/HTTPS keys.
- [ ] Master `03-basic-git-workflow`
  - Learn `git init`, `add`, `commit`, and `status`.

## Step 2: Working with GitHub (Week 2)
- [ ] Study `04-github-workflow`
  - Push your local code to GitHub, and pull updates.
- [ ] Learn `05-branching-and-merging`
  - Create branches, switch between them, and merge changes. Understand merge conflicts.

## Step 3: Collaboration & Open Source (Week 3)
- [ ] Dive into `06-collaboration`
  - Learn how to fork a repo and create a pull request (PR).
- [ ] Explore `08-github-features`
  - Write a good README, use `.gitignore`, and try GitHub CLI.

## Step 4: Advanced Git & Practice (Week 4)
- [ ] Understand `07-undo-and-fix`
  - Learn how to revert, reset, stash, and fix mistakes.
- [ ] Review `09-practical-workflows`
  - Memorize daily developer workflows.
- [ ] Complete `10-practice`
  - Finish all beginner and intermediate practice tasks!
