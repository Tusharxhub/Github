# Project Assignment

This is your final assignment to prove you have mastered the basics of Git and GitHub collaboration.

## The Open Source Simulation

1. **Find a Partner:** Find a friend who is also learning Git, or use a second GitHub account.
2. **Person A:** Creates a public repository called `git-collab-practice`. Add a simple `index.html` file with a heading. Push it to GitHub.
3. **Person B:** Forks Person A's repository.
4. **Person B:** Clones their fork to their computer.
5. **Person B:** Creates a new branch called `add-paragraph`.
6. **Person B:** Adds a paragraph of text to the `index.html` file, commits it, and pushes the branch to their fork.
7. **Person B:** Opens a Pull Request from their fork to Person A's original repository.
8. **Person A:** Reviews the code on GitHub, approves it, and clicks "Merge pull request".
9. **Person A:** Runs `git pull origin main` on their local computer to download Person B's changes.

If you can complete this workflow, you are ready to work on real software teams!
