# Beginner Tasks

Here is a checklist of beginner tasks to practice your basic Git skills.

## Tasks

- [ ] **1. Setup:** Create a GitHub account.
- [ ] **2. Install:** Install Git on your computer.
- [ ] **3. Configure:** Configure your Git username and email in the terminal.
- [ ] **4. Initialize:** Create a new folder on your computer and run `git init` inside it.
- [ ] **5. Create File:** Create a file named `README.md` and write a sentence in it.
- [ ] **6. Status:** Run `git status` to see the untracked file.
- [ ] **7. Add:** Run `git add README.md` to stage the file.
- [ ] **8. Commit:** Run `git commit -m "docs: add initial readme"` to save it.
- [ ] **9. GitHub:** Create an empty repository on GitHub.
- [ ] **10. Link:** Link your local repository to the GitHub remote using `git remote add origin <url>`.
- [ ] **11. Push:** Push your code to GitHub using `git push -u origin main`.

Congratulations! You just completed the fundamental Git workflow.
