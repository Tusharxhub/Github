# What is GitHub?

## What is it?
GitHub is a cloud-based hosting service that lets you manage Git repositories. While Git is the tool that tracks changes, GitHub is the website where you upload and share those tracked projects.

## Why do we use it?
We use GitHub to back up our code online, collaborate with other developers around the world, showcase our portfolio to employers, and contribute to open-source software.

## Quick Summary
- GitHub is a website that hosts Git repositories.
- It provides a visual interface for managing your code.
- Essential for teamwork and remote collaboration.
- Includes features like Pull Requests, Issues, and Actions.
