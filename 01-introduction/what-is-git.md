# What is Git?

## What is it?
Git is a distributed version control system created by Linus Torvalds. It runs locally on your computer and tracks changes to your project files.

## Why do we use it?
We use Git to safely develop software. If you make a mistake, you can always go back to a working version of your code. It also allows developers to work on new features in isolated "branches" without breaking the main codebase.

## Quick Summary
- Git runs on your local computer.
- It tracks every modification made to your code.
- You don't need an internet connection to use Git (until you want to share your code).
- It is free and open-source.
